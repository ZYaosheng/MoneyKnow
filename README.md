🎯 项目目标
在保留原 cashbook 核心记账功能的基础上，针对个人需求进行以下改造：

UI/UX 优化：调整界面布局和操作流程，适配个人使用习惯。

自动化增强：与 Tasker 联动，实现场景化记账（如定位到超市自动弹出食品分类）。

功能扩展：添加自定义报表、快捷记账小组件等实用功能。

## 使用说明
运行项目前请在项目 /gradle 路径下按如下格式添加文件 signing.versions.toml

```toml
[versions]
# 密钥别名
keyAlias = ""
# 别名密码
keyPassword = ""
# 密钥文件路径
storeFile = ""
# 密钥密码
storePassword = ""
```
